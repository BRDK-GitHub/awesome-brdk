# Awesome BRDK

Welcome to the **Awesome BRDK** repository.

This project is intended to be used by BRDK as a central hub for shareable developer content and resources.

## What's Inside?

This repository hosts a variety of tools and configurations designed to improve developer workflows, including:

*   **GitHub Copilot Resources**:
    *   **Custom Agents**: Specialized agents tailored for specific project needs.
    *   **Instructions**: Detailed instruction files (`.instructions.md`) to guide Copilot's context and generation.
    *   **Prompts**: Reusable prompt templates.
*   **Scripts**: A collection of utility scripts for various automation and development purposes.

## Structure

*   `copilot/`: Contains all Copilot-related definitions, agents, and instruction sets.
*   `brdk-scripts/`: General purpose scripts.

## Contributing

Feel free to contribute new agents, instructions, or scripts that can benefit the wider BRDK team.
